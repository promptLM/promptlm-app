# promptLM

This project is yet another **prompt** **L**ifecycle **M**anagement tool, but on the JVM.
It aims to not only help with storing, versioning, evaluating and distributing prompts, but also provides test support for mocked API endpoints for LLM inference server as well as MCP server.
This simplifies testing of AI applications and cuts costs testing.

## Features
- [ ] Create promptLM project (local and GitHub)
- [ ] Create promptLMSpec (save, execute and evaluate)
- [ ] Modify spec
- [ ] Release Prompt
- [ ] Publish prompt
- [ ] Publish test support
