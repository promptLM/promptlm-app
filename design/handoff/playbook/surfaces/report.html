Corpus report stable

A static, build-time-rendered overview of the prompt corpus. Three routes: overview, prompt detail, diff. Designed to be exported to HTML for sharing.

Data sources

The report is static, so all data is fetched once at build time and inlined. At runtime no API calls happen.
