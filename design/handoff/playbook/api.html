Release modes

Release flow has two server modes (direct vs pr_two_phase). The form's Release button (#95) calls updatePromptSpec() followed by releasePrompt() — see Release flow for the full state machine. Inspect the X-PromptLM-Release-State response header (requested | released) to decide whether to show "Release pending PR merge" follow-up UI that calls completeReleasePrompt(id, pr) after merge.
